Description détaillée

Modèle (pattern MVC) pour la gestion d'un Five or More.

Auteur
Lucas Lolo
Marzouk Ouro-Gomma
Version
1.0
Date
2/5/2021

Gestion du jeu five via une approche MVC. Ce module gère l'aspect modèle. Le modèle gère la valeur des grilles .

Par défaut, la grille est initialisé à 0.
