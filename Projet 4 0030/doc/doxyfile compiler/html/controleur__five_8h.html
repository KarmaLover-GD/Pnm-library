Description détaillée

Contrôleur (pattern MVC) pour la gestion d'un five.

Auteur
Lucas Lolo
Marzouk Ouro-Gomma
Version
1.0
Date
2/5/2021

Gestion d'un five via une approche MVC. Ce module gère l'aspect contrôleur. Le contrôleur fournit à l'utilisateur les contrôles sur le modèle ( En outre, le contrôleur traduit les actions de l'utilisateur en opération sur le modèle: déclencher les traitements par des appels de fonctions/procédures sur le modèle. Enfin, le contrôleur sélectionne et met à jour la vue.
